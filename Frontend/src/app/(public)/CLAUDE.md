# Frontend (public) Route Group
인증 없이 접근 가능하거나, 로그인/회원가입과 관련된 공용 경로들을 관리하는 디렉토리입니다.

## 주요 내용
- `/login`: 소셜 로그인 및 게스트 시작 진입점.
- `/onboarding`: 서비스 초기 정보 수집 (설문조사).
- `/privacy`, `/terms`: 법적 고지 페이지.

## 가이드라인
- 게스트 사용자도 온보딩 과정을 거쳐 대시보드 일부를 체험할 수 있도록 설계되어 있습니다.
- 환경 변수(`NEXT_PUBLIC_SUPABASE_URL`) 설정 오류 시 사용자에게 명확한 오류 안내를 노출합니다.
