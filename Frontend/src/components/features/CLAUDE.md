# Frontend Feature Components
도메인별 특화된 비즈니스 로직을 포함하는 컴포넌트들을 관리하는 디렉토리입니다.

## 주요 피처 구조
- `dashboard/`: 대시보드 위젯 및 요약 정보.
- `coach/`: AI 코칭 메시지 및 인터랙션.
- `log/`: 행동 로그 기입 및 타임라인 뷰.
- `survey/`: 온보딩 및 상태 체크 설문지.

## 가이드라인
- 각 피처 폴더 내부에 해당 피처에서만 쓰이는 `types.ts`나 `hooks/`를 둘 수 있습니다.
- 복잡한 피처의 경우 하위 폴더별로 `CLAUDE.md`를 추가하여 구조를 설명합니다.
- 비즈니스 로직은 가급적 `src/hooks/useQueries.ts` 등 전역 훅과 연동하여 처리합니다.
